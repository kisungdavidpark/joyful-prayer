## 1. 프로젝트 개요

### 1.1 목적
중보기도팀 조원들의 주간 기도 활동을 기록하고, 구글 폼으로 자동 제출하는 모바일 우선 웹앱.

### 1.2 주요 기능
- 일별 기도 시간 타이머/스톱워치 기록
- 성경통독 진행 체크
- 암송 구절 확인 및 녹음
- 새벽예배/금요예배/출석 보너스 시간 자동 반영
- 구글 폼 자동 prefill 및 제출
- 월간/연간 통계

### 1.3 주차 정의
- **화요일 00:00 ~ 월요일 23:59** 를 1주로 정의
- 주가 바뀌는 시점: **화요일 자정 (00:00)**
- 제출일: 다음 주 화요일

## 3. 앱 설치 방법

### Android (Chrome)
1. 브라우저에서 앱 URL 접속
2. 주소창 우측 **⋮ 메뉴** → **"홈 화면에 추가"** 선택
3. 또는 하단에 자동으로 뜨는 **"설치"** 배너 탭

### iOS (Safari)
1. **Safari**로 앱 URL 접속 (Chrome 불가)
2. 하단 **공유 버튼 (□↑)** 탭
3. **"홈 화면에 추가"** 선택
4. 우측 상단 **"추가"** 탭

> ⚠️ iOS는 반드시 Safari에서만 설치 가능합니다.

## 6. 탭별 기능 및 사용법

### 6.1 홈 탭

#### 이번 주 요약 카드 (4개)
각 카드를 탭하면 해당 탭으로 이동합니다.

| 카드 | 표시 내용 |
|------|------|
| 🙏 총 기도시간 | 이번 주 누적 기도 시간 / 1시간↑ 달성 일수 |
| 📖 통독 | 완료 장수 / 전체 장수 |
| ✍️ 암송 | 완료 / 미완 |
| ✅ 출석 | 출석/지각/조퇴/결석/미기록 |

#### 기도 파일
- 이번 주 기도 파일로 기도 여부 토글 (완료/미완)

#### 성령의 인도하심
- 이번 주 기도 중 받은 성령의 인도하심 메모

#### 출석 체크
| 선택 | 추가 입력 |
|------|------|
| 출석 | 없음 (중보모임 참석 시 +1시간 자동 반영) |
| 지각 | 지각 시간(분) + 사유 입력 |
| 조퇴 | 조퇴 시간(분) + 사유 입력 |
| 결석 | 결석 사유 입력 |

> 출석/지각/조퇴 선택 시 해당 주 화요일 기도 시간에 **+1시간** 자동 반영  
> 결석 선택 또는 해제 시 자동 차감

#### 주간 제출
1. **📤 제출** 버튼 클릭 → 입력 내용 확인(confirm) → 구글 폼 자동 열림
2. 폼 확인 후 제출
3. 제출 완료 후 **텍스트 복사** / **📨 공유** 버튼 활성화
4. 공유 텍스트를 카카오톡 등으로 공유

> ⚠️ 텍스트 복사/공유는 **구글 폼 제출 완료 후에만** 가능합니다.

---

### 6.2 기도 탭

#### 타이머 / 스톱워치
| 기능 | 설명 |
|------|------|
| 스톱워치 | 기도 시간 측정 후 저장 |
| 타이머 | 목표 시간 설정 후 카운트다운 |

**타이머 시간 설정 방법**
- 원형 디스플레이 내 시간 표시를 **위아래 드래그** 또는 **마우스 휠**로 조절
- 시간: 0~5시간, 분: 0~55분 (5분 단위)

**타이머 완료 알림**
- 📳 진동 (Android)
- 🔔 알림음 (종소리 3번)
- 🔔 브라우저 알림 (백그라운드 시)

> 타이머 실행 중 다른 탭으로 이동해도 계속 카운트됩니다.

#### 요일 선택 바
- 화~월 7개 요일 중 기록할 날짜 선택
- 오늘 날짜 강조 표시
- 1시간 이상 달성 시 초록색 표시

#### 새벽예배 / 금요HR예배

| 구분 | 보너스 | 입력 방법 |
|------|------|------|
| 새벽예배 | 참석일 +1시간 | 요일별 체크 |
| 금요HR예배 (11시까지) | +1시간 | `~11시` 버튼 |
| 금요HR예배 (12시까지) | +2시간 | `~12시` 버튼 |

> 보너스 적용 후 **수정 버튼**으로 전체 시간 자유 변경 가능

#### 기도 시간 수동 수정
- 요일 카드의 **수정** 버튼 클릭
- 시간/분 드럼롤(스크롤)로 직접 입력

---

### 6.3 통독 탭

- 이번 주 통독 범위 표시 (책명 + 장 목록)
- 장별 체크박스로 개별 완료 체크
- **성경 전체통독 완료** 별도 체크 버튼

> 통독 데이터는 `schedule.json`에서 관리합니다.

---

### 6.4 암송 탭

#### 암송 구절 표시
- **이전 주 최신 구절 1절** + **이번 주 구절** 함께 표시
- 이전에 데이터가 있는 가장 최근 주의 마지막 절 자동 표시

#### 암송 녹음
1. **● 녹음 시작** 버튼 클릭 (마이크 권한 필요)
2. 암송 후 **■ 녹음 중지**
3. 재생 배속 선택: 0.5x ~ 2x

#### 암송 완료 체크
- **암송 완료** 버튼 체크
- 틀린 글자 수 선택 (0~10자)

| 틀린 글자 수 | 폼 입력값 |
|------|------|
| 0자 | 1 (완료) |
| 1~3자 | 0.5 |
| 4자 이상 | 0 |
| 미완료 | 0 |

---

### 6.5 통계 탭

#### 주간
- 주차별 기도 시간, 기도 일수, 새벽기도, 통독, 제출 여부

#### 월간
- 월별 총 기도시간, 기도 일수, 새벽기도, 통독 장수, 완독, 암송 구절 수

#### 연간
- 연도 선택 버튼 (올해 🔸 강조)
- 선택 연도 상세 통계
- 주평균 기도시간

> 암송 구절 수는 **중복 제외** 실제 신규 절 수만 카운트합니다.

## 7. 구글 폼 자동 제출

### 7.1 제출 절차
1. 홈 탭 → **출석 체크** 선택 (필수)
2. **📤 제출** 버튼 클릭
3. 입력 내용 확인(confirm 팝업) → **확인**
4. 구글 폼이 자동으로 열리며 데이터 자동 입력
5. 폼에서 최종 **제출** 클릭
6. 제출 완료 후 텍스트 복사/공유

### 7.2 제출 검증 항목
| 항목 | 조건 |
|------|------|
| 출석 체크 | 미선택 시 제출 불가 |
| 지각 | 지각 시간 + 사유 필수 |
| 조퇴 | 조퇴 시간 + 사유 필수 |
| 결석 | 결석 사유 필수 |

### 7.3 폼 입력 항목 매핑

#### 목회자중보 폼
| 항목 | 내용 |
|------|------|
| 조 | 프로필 조 선택값 |
| 날짜 | 제출일 (년/월/일 분리 입력) |
| 이름 | 프로필 이름 |
| 출석 | 1 (출석/지각/조퇴) / 0 (결석) |
| 파일기도 | 1/0 |
| 성경통독 | 1/0 |
| 성령인도하심 | 1/0 |
| 성경암송 | 1/0.5/0 |
| 매일기도 | 0~6 (1시간 이상 기도한 일수) |
| 총기도시간 | 정수 (시간 단위) |
| 지각 감점 | -0.5 (지각 시) |
| 사유 | 지각/조퇴/결석 사유 |
| 전체통독 | 1독 완료 |

#### 교회중보 폼
목회자중보와 동일하되, **지각/조퇴 감점이 별도 항목**으로 분리:
- 지각: `lateCheck` (-0.5)
- 조퇴: `lateCheck2` (-0.5)

## 8. 데이터 관리

### 8.1 데이터 저장 위치
모든 기도 기록은 **브라우저 localStorage**에 저장됩니다.

| 키 | 내용 |
|------|------|
| `profile` | 중보 유형, 조, 이름 |
| `week_YYYY-MM-DD` | 주간 기도 기록 |
| `easyMode` | 쉬운모드 설정 |
| `scheduleCache` | schedule.json 오프라인 캐시 |

> ⚠️ "인터넷 사용 기록 삭제"에서 **쿠키 및 사이트 데이터**를 지우면 모든 기록이 삭제됩니다.  
> 일반 캐시 삭제(앱 업데이트)와는 무관합니다.

### 8.2 데이터 백업 / 복원
**설정 탭 → 💾 데이터 백업/복원**

| 기능 | 방법 |
|------|------|
| 내보내기 | 📥 내보내기 버튼 → JSON 파일 다운로드 |
| 가져오기 | 📤 가져오기 버튼 → 백업 파일 선택 → 자동 복원 |

- 파일명: `joyful-prayer-backup-YYYY-MM-DD.json`
- 기기 교체, 앱 재설치 후 복원 시 활용
