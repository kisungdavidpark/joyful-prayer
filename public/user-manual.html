1. 처음 시작
- 중보 유형 선택
- 조 선택
- 이름 입력 후 시작
2. 홈 화면
- 이번 주 기도시간 확인
- 요일별 기도 여부 확인
- 통독 / 암송 / 출석 상태 확인
3. 기도 탭
- 기도 시작 → 시간 자동 기록
- 종료 시 저장
- 수정 버튼으로 직접 입력 가능
4. 통독 탭
- 읽은 장 클릭하여 체크
- 전체체크로 한 번에 완료 가능
5. 암송 탭
- 암송 완료 체크
- 틀린 글자 수 선택 (0~5)
6. 제출
- 출석 상태 선택
- 제출 버튼 클릭 → 구글폼 이동
7. 설정
- 쉬운모드 (글자 크기 변경)
- 데이터 백업 / 복원
