🙏 중보기도 기록앱 사용자 매뉴얼

처음 사용하는 분을 위한 화면별 핵심 사용법
첫 실행홈기도통독암송통계제출설정

1. 처음 실행할 때

첫 화면
중보 유형 선택 → 조 선택 → 이름 입력 → 시작하기
교회중보목회자중보
  1. 중보 유형을 먼저 선택합니다.
  2. 본인의 조와 이름을 입력합니다.
  3. 시작하기를 누르면 홈 화면으로 이동합니다.

iPhone은 Safari에서 공유 버튼(□↑) → 홈 화면에 추가를 사용하면 앱처럼 실행할 수 있습니다.

2. 홈 탭

이번 주 전체 현황
총 기도시간 / 기도일수 / 통독 / 암송 / 출석을 한눈에 확인합니다.
화 ✓수 ·목 -금 🔥
  • 기도 파일: 이번 주 기도 파일로 기도했으면 완료로 바꿉니다.
  • 성령의 인도하심: 기도 중 받은 내용을 짧게 기록합니다.
  • 출석 체크: 출석/지각/조퇴/결석을 선택합니다.

출석/지각/조퇴를 선택하면 해당 주 화요일 기도시간에 +1시간이 반영됩니다. 그 상태에서 결석으로 바꾸면 -1시간이 됩니다.

3. 기도 탭

기도시간 기록
기도 시작 → 기도 후 종료
시간은 1분 단위로 자동 누적됩니다.

기도시간 수정

  1. 아래 주간 기도 기록에서 수정할 요일의 수정을 누릅니다.
  2. 시간/분을 선택합니다.
  3. 저장을 누르면 해당 요일 시간이 변경됩니다.

예배 출석 반영

  • 금요HR예배: 11시까지 +1시간, 12시까지 +2시간을 선택합니다.
  • 새벽예배: 월~금은 카운트 +1 / 시간 +1, 토요일은 시간만 +1, 일요일은 선택할 수 없습니다.

4. 통독 탭

이번 주 통독
이번 주 읽을 성경 장이 표시됩니다.
1장2장3장
  • 읽은 장을 하나씩 누르면 체크됩니다.
  • 전체체크로 한 번에 완료할 수 있습니다.
  • 성경 전체 1독을 완료했으면 성경 전체 1독도 체크합니다.

5. 암송 탭

암송 대상
갈라디아서 5:23
온유와 절제니...
녹음 시작으로 암송을 녹음하고 들어볼 수 있습니다.
  • 암송 대상은 전주 대상이 있으면 함께 표시됩니다.
  • 암송 후 암송 완료하기를 누릅니다.
  • 틀린 글자 수는 0~5 중에서 선택합니다.

6. 통계 탭

주간 / 월간 / 연간 통계
주간월간연간
  • 주간: 각 주의 기도, 통독, 새벽기도, 제출 여부를 확인합니다.
  • 월간: 월별 누적 기도시간과 통독 장수를 확인합니다.
  • 연간: 연간 누적 기록을 확인합니다.

7. 주간 제출

홈 탭 하단
텍스트 복사 / 공유 / 제출
  1. 홈 탭에서 입력 내용을 확인합니다.
  2. 필요하면 공유 텍스트 미리보기를 열어 확인합니다.
  3. 제출을 누르면 구글폼이 열립니다.

지각/조퇴/결석은 사유 입력이 필요합니다.

8. 설정 탭

  • 쉬운모드: 기본/크게/더 크게/아주 크게로 폰트 크기를 조절합니다.
  • 내 정보: 중보 유형, 조, 이름을 변경합니다.
  • 데이터 백업/복원: 기록을 파일로 저장하거나 복원합니다.
  • 사용자 매뉴얼: 이 문서를 다시 열 수 있습니다.
